## 🛠️ 3. เครื่องมือหลักที่ใช้และเหตุผล
**เทคโนโลยีที่เลือกใช้:** `Flutter`

## 🔗 4. ความสำคัญต่อภาพรวมของกลุ่ม
งานฝั่ง **Frontend** ส่งผลกระทบโดยตรงต่อความสำเร็จของโครงการ เพราะ:
1. **User Experience:** หากออกแบบและพัฒนาได้ดี จะช่วยให้ผู้ใช้งานเข้าใจระบบได้ง่ายและใช้งานสะดวก
2. **The Bridge:** เป็น "ตัวกลาง" ในการเชื่อมต่อระหว่างผู้ใช้กับระบบหลังบ้าน (Backend และ Blockchain)
🚨 **ข้อควรระวังสำหรับทีม:** หากเกิดข้อผิดพลาดในส่วนนี้ (เช่น การแสดงผลข้อมูลไม่ถูกต้อง, หรือเรียกใช้งาน API ผิดพลาด) จะส่งผลให้ภาพรวมของระบบทำงานไม่สมบูรณ์
